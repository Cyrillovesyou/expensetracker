A simple web application app for tracking daily expenses by category.

Features
-Add and delete expenses 
-Category filtering
-Automatic total calculation
-Responsive Design

Technologies Used
-HTML
-CSS
-Javascript

Live Demo:  https://cyrillovesyou.github.io/expensetracker/
